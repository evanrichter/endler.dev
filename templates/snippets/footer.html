About me

I'm a Backend Engineer running Hello, Rust!, a YouTube channel about the Rust programming language, codeprints.dev, a shop for printing your GitHub timeline, and analysis-tools.dev, an open platform for static code analysis tools. On this blog, you'll mostly find articles about programming in Rust and Golang or developer tools like make and Vim. Don't know where to start? Check out the blog archive or my about page.
