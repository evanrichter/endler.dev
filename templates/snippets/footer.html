About me

I'm Matthias Endler, a Backend Engineer at corrode, helping clients make their Rust code faster and more idiomatic. I maintain analysis-tools.dev, an open platform for static code analysis tools and Idiomatic Rust. Previously, I ran Hello, Rust!, a YouTube channel about the Rust programming language and codeprints.dev, a shop for printing your GitHub timeline. For more info, check out the about page or my blog archive.
